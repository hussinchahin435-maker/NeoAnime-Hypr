# 🌌 Celestia (NeoAnime-Hypr)

واجهة احترافية مخصصة ومبنية فوق بيئة **Hyprland** و **EndeavourOS** بتصميم مستوحى من عالم الأنمي.

## ✨ المميزات
- تثبيت ذكي بنظام رسومي (Zenity).
- دعم كامل للغتين العربية والإنجليزية.
- إعداد تلقائي لتخطيط الكيبورد مع اختصار للتبديل (`Win + Space`).
- شريط مهام تفاعلي (Waybar) يحتوي على مؤشر للغة النظام.
- خلفية أنمي مخصصة مدمجة تلقائياً.

## 🛠️ طريقة التثبيت
بعد تثبيت نظام EndeavourOS النظيف (Minimal Install)، افتح الترمينال واكتب الأمر التالي:

```bash
curl -sL [https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/ctl](https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/ctl) -o ctl && chmod +x ctl && ./ctl
